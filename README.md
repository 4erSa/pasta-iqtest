## 🚀 Как запустить это чудо

Проект разделен на две части: **Сайт** и **Сервер уведомлений**.

### 1. Сервер (`/server`)
Сервер управляет уведомлениями и удаленным запуском видео.
```bash
cd server
npm install
node server.js
```
**Команды в консоли сервера:**
*   `/notify Заголовок/Текст` — отправить системное сообщение.
*   `/vid 5s` — запустить рикролл на 5 секунд.
*   `/exit` — выключить сервер.

### 2. Сайт (`/site`)
Сам тест на React.
```bash
cd site
npm install
npm run dev -- --host
```
После запуска открой `http://localhost:5173`.
